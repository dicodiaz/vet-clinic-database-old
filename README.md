# Vet Clinic Database

This is a project that uses a relational database to create the data structure for a vet clinic. The complete database includes data about:

- Animals
- Animals' owners
- Clinic employees
- Visits

## Built With

- PostgreSQL
- VSCode
- Git & GitHub
